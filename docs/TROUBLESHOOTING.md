# Troubleshooting\n\nCommon issues and solutions.\n\n## Invoice Not Showing\n\nIf an invoice doesn't appear in the table:\n1. Check the status filter — you may be filtering by Paid/Pending\n2. Refresh the page\n3. Verify the invoice was saved successfully\n\n## API Returning 500\n\nIf the API returns a 500 error:\n1. Check the server logs\n2. Verify the database connection\n3. Ensure the query parameters are valid
\n\n## Export Issues\n\nIf CSV export fails, check that the data set is not empty before exporting.
